# student-management-system-jdbc
The Student Management System is a database-driven console application built using Core Java, JDBC, and SQL. It helps manage students, courses, enrollments, and grades in an organized way.

Features

1.Student Module
Student Registration
Update Student Profile
Enroll in Courses
View Available Courses
View Grades
View Profile Details

2.Admin Module
Add Course
Update Course
Delete Course
View All Students
View All Courses
View All Enrollments
Assign/Update Grades
Add Student, Course, and Enrollment together

3.System Features
Batch Execution for inserting multiple courses
Batch Execution using Stored Procedures for adding multiple students
Database transaction handling
Menu-driven console application

Tech Stack
Language: Java
Database Connectivity: JDBC
Database: PostgreSQL / MySQL
SQL Concepts: Joins, Procedures, Batch Execution, Transactions
IDE: Eclipse / IntelliJ

Key Concepts Used
Object-Oriented Programming (OOP)
JDBC API
PreparedStatement
CallableStatement
ResultSet Handling
Batch Processing
Stored Procedures
Exception Handling

Project Objective

To build a real-world academic management system for handling student records, course management, enrollment processes, and grading with efficient database operations.
